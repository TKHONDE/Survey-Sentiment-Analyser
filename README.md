1. A Python-based tool that analyzes survey responses and classifies them as positive, negative, or neutral using NLP libraries like spaCy and TextBlob.

2. FEATURES:

Reads CSV survey data.
Cleans and processes text.
Performs sentiment analysis.
Generates word clouds and frequency plots.
Outputs categorized results.

3. TECHNOLOGIES USED:

Python
Libraries: spaCy, TextBlob, Pandas, Matplotlib, WordCloud

4. FUTURE IMPROVEMENTS:

Add support for larger spaCy models (en_core_web_md, en_core_web_lg).
Multi-language sentiment analysis.
Interactive dashboard for results.
